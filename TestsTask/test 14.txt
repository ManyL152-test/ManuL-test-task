Операция сложения двух чисел (позитивный сценарий)
Цель: проверить корректную работу функции сложения.
Шаги:
ввести два числа через запятую;
нажать кнопку «+»;
проверить результат в поле вывода.
Ожидаемый результат: правильный результат сложения двух введённых чисел.

Негативный сценарий: ввести нечисловые символы перед числами или после нажатия кнопки «+». Ожидаемый результат: сообщение об ошибке или некорректный результат.

Вычитание одного числа из другого (позитивный сценарий)
Цель: протестировать функцию вычитания.
Шаги:
ввести уменьшаемое и вычитаемое через запятую;
нажать кнопку «−»;
проверить результат.
Ожидаемый результат: корректный результат вычитания второго числа из первого.

Негативный сценарий: попытаться вычесть большее число из меньшего. Ожидаемый результат: предупреждение о невозможности выполнения операции или корректный вывод сообщения об ошибке.

Умножение двух чисел (негативный сценарий)
Цель: выявить ошибки при выполнении умножения.
Шаги:
ввести два случайных числа;
умножить их, нажав кнопку «×»;
обратить внимание на результат.
Ожидаемый результат: либо корректный результат умножения, либо сообщение об ошибке, если одно из чисел равно нулю.

Позитивный сценарий: перемножить два положительных числа. Ожидаемый результат: корректный результат умножения.

Деление двух чисел (позитивный сценарии)
Цель: убедиться в правильной работе функции деления.
Позитивный сценарий (шаг 1):
ввести делимое и делитель через запятую;
разделить их, нажав кнопку «÷»;
проверить результат.
Ожидаемый результат: корректный результат деления первого числа на второе, если второе число не равно нулю.
Позитивный сценарий (шаг 2):
повторить шаги для случая, когда делитель равен единице.
Ожидаемый результат: результат должен быть равен введённому делимому числу.

Негативный сценарий: попробовать разделить на ноль. Ожидаемый результат: появление сообщения об ошибке.

Работа с памятью (позитивный сценарий)
Цель: оценить функционал памяти калькулятора.
Позитивный сценарий:
использовать кнопки памяти для сохранения и вызова значений;
выполнить арифметическую операцию с использованием сохранённого значения;
убедиться, что результат соответствует ожидаемому.
Ожидаемый результат: калькулятор корректно использует сохранённое значение для выполнения операций.

Негативный сценарий: попытаться сохранить или вызвать несуществующее значение памяти. Ожидаемый результат: калькулятор выдаёт ошибку или не выполняет операцию.
